## 四個主屬性

角色成長配點與裝備加成都作用在這四個主屬性上（`attrs = {str, agi, int, luck}`）：

| 主屬性 | 中文 | 主要影響 |
|---|---|---|
| `str` | 力量 | HP 上限、物防；若為主屬性還撐物攻 |
| `agi` | 敏捷 | 閃避、命中、會心、行動力(ATB 速度)；若為主屬性還撐物攻 |
| `int` | 智力 | MP 上限、魔攻、魔防；若為主屬性還撐物攻 |
| `luck` | **幸運**（v4.0 新增）| 會心、抗爆、閃避、掉寶率 |

- **主屬性 `mainAttr`**：每個角色一個（str/agi/int），物攻 `patk = weaponAtk + 主屬性值×2 + 裝備patk`。
- **裝備也能加主屬性**（v4.0）：裝備 `stats` 放 `str/agi/int/luck` 會先疊進「有效屬性」再算所有衍生值
  （舊版只讀角色 base、裝備加主屬性不生效——這是本次修的核心落差）。

## 衍生戰鬥數值（由主屬性＋裝備算出，`derive()`）

| 衍生值 | 中文 | 公式（有效屬性；係數見下） |
|---|---|---|
| `maxhp` | HP 上限 | `hpBase + STR×hpPerStr + 裝備hp` |
| `maxmp` | MP 上限 | `mpBase + INT×mpPerInt + 裝備mp` |
| `patk` | 物攻 | `weaponAtk + 主屬性×2 + 裝備patk` |
| `matk`/`pdef`/`mdef` | 魔攻/物防/魔防 | 見 spec F-1 |
| `dodgeV` | 閃避 | `round(AGI×dodgePerAgi + LUK×dodgePerLuck) + 裝備dodge` |
| `accV` | 命中 | `round(AGI×accPerAgi) + 裝備acc` |
| `critV` | 會心(%) | `round((critBase + AGI×critPerAgi + LUK×critPerLuck + 裝備crit)×10)/10` |
| `critresV` | 抗爆(%) | `round(LUK×critresPerLuck×10)/10 + 裝備critres` |
| `critdmg` | 爆傷倍率 | `critDmgBase + 裝備critdmg`（爆擊時傷害×此值）|
| `spd` | 行動力 | `AGI + 裝備spd`（決定 ATB 蓄力速度）|

## 一次攻擊拆三段（普攻與敵人單體技能）

1. **命中**：`命中率% = 100 − clamp(閃避守 − 命中攻, 0, dodgeCap)`；未命中＝MISS，結束。
2. **爆擊**：`有效會心% = clamp(會心攻 − 抗爆守, 0, critCap)`；擲骰決定是否爆擊。
3. **傷害**：`max(1, (物攻×1.8 − 物防)[防禦中×0.5] × 隨機0.85~1.0 × (爆擊? critdmg : 1))`。

> 技能傷害（玩家 int/物理技）與全體攻擊**不判命中**（刻意設計，見 spec F-4/F-8）。

## 係數表（`derived.tres` / content_db 內嵌 `derived`）

| 係數 | 值 | 意義 |
|---|---|---|
| `critPerAgi` | 0.2 | 1 敏捷 → 會心 +0.2% |
| `critPerLuck` | 0.1 | 1 幸運 → 會心 +0.1% |
| `dodgePerAgi` | 1.5 | 1 敏捷 → 閃避值 +1.5 |
| `accPerAgi` | 1.5 | 1 敏捷 → 命中值 +1.5（＝dodgePerAgi，故敏捷相等時淨閃避＝雙方敏捷差×1.5）|
| `dodgePerLuck` | 0.05 | 1 幸運 → 閃避值 +0.05 |
| `critresPerLuck` | 0.1 | 1 幸運 → 抗爆 +0.1% |
| `critDmgBase` | 1.4 | 爆擊傷害基礎倍率（裝備 critdmg 疊加）|
| `dropPerLuck` | 0.1 | 1 幸運 → 掉寶率 +0.1%（取全隊最高 luckV，F-10）|
| `critCap` | 100 | 有效會心率上限(%) |
| `dodgeCap` | 30 | 淨閃避上限(%) |
| `critBase` | 6.25 | 基礎會心率(%)（敵人一般怪即此值）|

## 起始幸運值（可在 Inspector 調）

- 隊員 base luck：路德 2、瑪琳 3、亞倫 8。
- 敵人 luck：一般怪 0（行為與 v3.0 相同）；boss——哥布林頭目 3、狂暴洞熊 3、洞窟食人魔 4、異變的魔影 5。
- 敵人 luck 影響：會心（`critBase + luck×critPerLuck`）、抗爆（`luck×critresPerLuck`）、閃避（`+luck×dodgePerLuck`）。

## 怎麼調（設計員）

- **改係數**：Godot 編輯器開 `content_db.tres` → `derived` 子資源改欄位（runtime 真相源）；同步改
  `resources/content/derived.tres`（種子）與本檔＋spec。
- **改角色/敵人 luck**：開 `content_db.tres` 對應 party/enemy 子資源改 `base.luck` / `luck`；同步個別種子 .tres。
- **爆傷養成**：給武器/飾品的 `stats` 加 `critdmg`（如 0.3 → 爆擊 ×1.7）、`acc`（命中）、`critres`（抗爆）。
- ⚠️ 改 `.tres` 個別檔對 runtime 無效——真相源是 `content_db.tres` 內嵌 SubResource（見
  [CLAUDE.md](../../CLAUDE.md)／memory `content-data-pipeline`）。
