## 刀 部件說明
| 編號 | 中文  | 英文                  | 說明                              |
| -- | --- | ------------------- | ------------------------------- |
| 1  | 刀尖  | Tip / Point         | 刀最前端，用於穿刺與精確攻擊。                 |
| 2  | 刀腹  | Belly               | 刀刃具有弧度的部分，主要用於切割。               |
| 3  | 假刃  | False Edge / Swedge | 刀背前端削薄的部位，可提升穿刺能力，部分刀具會開鋒。      |
| 4  | 刀脊  | Spine               | 刀身背部，未開鋒，提供刀身強度。                |
| 5  | 刀刃  | Edge                | 鋒利的切割面，是主要攻擊部位。                 |
| 6  | 血槽  | Fuller              | 刀身上的凹槽，可減輕重量並提升結構強度（並非放血用途）。    |
| 7  | 刀肩  | Plunge              | 刀刃與刀身側之間的過渡位置，形成研磨起點。           |
| 8  | 刀身側 | Ricasso             | 靠近護手、未開鋒的刀身區域，常刻有品牌或鍛造標記。       |
| 9  | 刀頸  | Choil               | 刀刃末端與刀柄之間未開鋒的凹位，可供手指施力，提升精細操作性。 |
| 10 | 護手  | Guard               | 保護手部，防止握持時滑向刀刃，也可抵擋敵方武器。        |
| 11 | 刀柄  | Handle / Grip       | 握持部位，通常包覆皮革、木材或其他防滑材質。          |
| 12 | 柄尾  | Pommel              | 刀柄尾端，用於配重，部分設計可作為敲擊工具。          |

**刀身 (Blade): 1 - 9 項合起來的簡稱**

## 角色立繪 / 武器立繪

製作此類產物時，可以完整呈現所有部件的特色

## 戰鬥立繪 / 角色互動立繪
- 製作此類產物時，考慮到畫風與尺寸的呈現方式，只強調 刀身、刀頸、護手、刀柄、柄尾。
- 刀完整的長度約為角色頭頂到腳底高度的35%，刀身的長度為刀柄的 4 倍。
