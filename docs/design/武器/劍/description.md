## 刀 部件說明
| 編號 | 中文 | 英文                 | 說明                          |
| -- | -- | ------------------ | --------------------------- |
| 1  | 劍尖 | Tip / Point        | 劍最前端，用於刺擊。                  |
| 2  | 劍刃 | Edge               | 鋒利的切割面，主要攻擊部位。              |
| 3  | 劍脊 | Spine              | 劍身背面，提供結構強度。                |
| 4  | 劍身 | Blade              | 劍柄以外的整個金屬部分。                |
| 5  | 血槽 | Fuller             | 劍身中央凹槽，可減輕重量並提升結構強度（不是放血用）。 |
| 6  | 劍格 | Ricasso            | 劍格前方未開鋒的區域，增加結構強度。          |
| 7  | 護手 | Guard / Crossguard | 保護手部，防止滑向劍刃，也可格擋敵方武器。       |
| 8  | 劍柄 | Grip / Handle      | 握持部位，通常包覆皮革或纏繩。             |
| 9  | 柄首 | Pommel             | 劍柄尾端，用來配重，使整把劍更平衡。          |

## 角色立繪 / 武器立繪
