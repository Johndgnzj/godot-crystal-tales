## 動畫 Strip（產 strip 時才檢查）

- [ ] 面向正確（我方左、敵方右；只產基準向，翻轉得另一向）
- [ ] attack 武器完整在格內
- [ ] idle 循環 3→0 無縫、不漂移
- [ ] 每幀腳底基準線一致（root-lock）
- [ ] 每個輸出幀都保留該幀角色、武器與特效的完整輪廓，不得被格線裁切
- [ ] 每個輸出幀不得包含前一幀或下一幀的任何角色、武器、特效像素
- [ ] 逐幀檢查透明背景結果；幀與幀之間及畫布外圍必須保持乾淨透明留白

### Strip 預覽包（正式整合前必須具備）

- [ ] 保留原始 strip，並可清楚辨識每個幀的預定範圍
- [ ] 已輸出固定尺寸的逐幀 PNG，且每幀可獨立檢查
- [ ] 已輸出標示幀號的 Review montage，並逐幀查看角色、武器與特效
- [ ] 原始 strip 與逐幀 PNG 都通過檢查；不可只查看 montage
- [ ] 若切格邊界不明確，已人工修正或退回重產，不得直接採用等寬硬切結果
- [ ] 預覽驗收通過前，不得複製到 `godot-project/assets/`
