## 可動畫化

- [ ] 左右輪廓清楚
- [ ] 武器完整
- [ ] 手腳沒有遮擋
- [ ] 不會影響 Idle
- [ ] 不會影響 Attack
- [ ] 持劍手的拇指與四指完整環握劍柄；手腕與劍柄同軸，沒有穿掌、夾持或反向握劍
- [ ] 雙手劍的雙手皆完整握住可容納雙手的劍柄，沒有手指重疊、穿模或武器變形
- [ ] cast 的視線、施法手勢與武器尖端（若使用）都朝敵方；沒有光球、符文、粒子、光暈或發射物
- [ ] 選背面 cast 時為三分之二背面，施法手、武器與部分側臉不被披風遮住
- [ ] 選倒下 death 時，武器自然掉落在原持手附近、完整留在畫布內，沒有額外物件

- [ ] 面向正確（我方左、敵方右；只產基準向，翻轉得另一向）
- [ ] attack 武器完整在格內
- [ ] idle 循環 3→0 無縫、不漂移
- [ ] 每幀腳底基準線一致（root-lock）
- [ ] 每個輸出幀都保留該幀角色、武器與特效的完整輪廓，不得被格線裁切

- [ ] 逐幀檢查透明背景結果；幀與幀之間及畫布外圍必須保持乾淨透明留白
