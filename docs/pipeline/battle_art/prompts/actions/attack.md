# attack 動作資料集

## 選項

| ID | 招式 | prompt 動作文字 |
|---|---|---|
| `sword_diagonal_slash` | 斜劈 | 主手持劍由後上方蓄力，沿斜向朝敵方揮出；肩、手肘、手腕與劍柄同向帶動。 |
| `sword_horizontal_slash` | 水平斬 | 主手持劍以腰肩旋轉帶動，劍尖沿朝敵方的水平弧線揮出。 |
| `sword_thrust` | 突刺 | 主手直線推出劍柄，劍尖朝敵方，後腳支撐、身體前傾但腳底基準穩定。 |
| `two_hand_sword_overhead` | 雙手上劈 | 雙手一上一下完整握住可容納雙手的長劍柄，由後上方蓄力，朝敵方斜向劈下；兩手、劍柄與手腕方向一致。 |
| `dagger_thrust` | 匕首突刺 | 主手以短促直線刺向敵方，手肘收束、後腳支撐，匕首不可變成長劍。 |
| `staff_strike` | 杖擊 | 雙手穩定握住杖身，以肩與腰帶動杖端朝敵方擊出；杖長與握點維持 seed 定義。 |
| `claw_swipe` | 爪擊 | 身體前傾，前爪沿朝敵方的短弧線揮出；爪數、長度與手部輪廓維持 seed 定義。 |

## 基準分鏡

`0 起手 → 1 蓄力 → 2 揮出／刺出 → 3 命中頂點 → 4 收招`

## 共通硬限制

- 每格武器完整留在格內；攻擊弧朝敵方（我方角色左、敵方角色右），不可改變面向或讓武器跨格。
- 武器、爪或肢體的種類、尺寸、配色與 seed 完全一致，不可在動作中替換或變形。

## 單手劍額外硬限制

- 選 `sword_*` 時，主手、劍長、劍身／劍柄比例、護手與配色必須與 seed 完全一致。
- 拇指與四指完整環握劍柄；手腕與劍柄同軸。劍柄不可穿過手掌、不可只由中指或無名指夾持，也不可出現反向握法。

## 雙手劍額外硬限制

- 只可用於 seed 已定義為雙手劍或劍柄足以容納雙手的武器；單手劍不得強行改成雙手握法。
- 兩手必須都以拇指與四指完整環握劍柄，兩手位置不得重疊、穿模或遮住彼此；劍長與握柄長度維持 seed 定義。
