# cast 動作資料集

## 選項

| ID | 施法方式 | prompt 動作文字 |
|---|---|---|
| `forward_palm` | 單手前伸 | 非持武器手朝敵方前伸、掌心與指尖指向敵方，視線沿手勢望向敵方；不描繪任何法術效果。 |
| `weapon_channel` | 武器引導 | 主手穩定持武器，武器尖端朝敵方引導法術；視線與法術方向一致。 |
| `two_hand_focus` | 雙手蓄能 | 雙手在胸前作蓄能手勢，身體與視線仍朝敵方；不描繪任何魔力或釋放特效。 |
| `offhand_rear_cast` | 副手背面施法 | 身體採三分之二背面，披風朝鏡頭；非持武器手從身側朝敵方伸出，仍清楚看見手勢、主手武器與部分側臉。 |
| `weapon_channel_rear` | 武器背面引導 | 身體採三分之二背面，披風朝鏡頭；以慣用手持武器朝敵方引導施法，武器尖端、視線與身體方向一致，保留部分側臉。 |

## 硬限制

- 我方角色敵方方向是畫面左側；敵方角色敵方方向是畫面右側。
- 施法手、眼神與武器尖端（若使用武器引導）必須朝同一方向；表情專注、有戰鬥張力，不可望向鏡頭。
- 持劍者未選 `weapon_channel` 時，主手仍須以 seed 的正確握法穩定持劍或自然收劍。
- **不產生魔法效果**：禁止光球、符文、粒子、光暈、發射物與任何漂浮特效；施法意圖只以姿勢、視線與手勢／武器方向表現，特效由 Godot 另行製作。
- `*_rear` 僅用於明確選擇背面構圖的角色；必須是三分之二背面，不可全背面，且施法手、武器與部分側臉不可被披風遮住。
