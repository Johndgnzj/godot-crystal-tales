# death 動作資料集

## 選項

| ID | 死亡方式 | prompt 動作文字 |
|---|---|---|
| `fall_forward` | 向前倒下 | 重心已失去支撐、身體朝前方倒下，武器自然掉落於原持手附近，輪廓仍可辨識。 |
| `fall_backward` | 向後倒下 | 背部先失去支撐而向後倒下，四肢自然鬆開，武器掉在原持手一側，輪廓仍可辨識。 |
| `kneel_collapse` | 跪地潰倒 | 單膝或雙膝失力下沉、上半身垂落，武器不憑空消失。 |
| `dissolve` | 魔法潰散 | 身體由邊緣少量粒子化潰散，主輪廓仍清楚，不做大面積特效遮蔽。 |

## 硬限制

- death 是單獨定格，不得混用 hurt 或 idle 姿勢。
- 裝備與武器必須仍可辨識，除非模板明確為潰散效果。選擇倒下模板時，武器應自然**掉落**於原持手附近、完整留在畫布內；它是角色原持有物，不可生成額外物件。
