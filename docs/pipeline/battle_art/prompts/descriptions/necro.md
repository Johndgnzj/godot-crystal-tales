- 最後一版: 2026-07-22 首幀組（HD Pixel 二頭身，John 已驗收）

```
約 32 歲男性死靈術士，蒼白膚色，黑色中長髮部分遮住臉；深黑兜帽、黑色與深紫色破碎長袍及層疊披風、暗色腰帶、露指手套與黑色靴子，胸前有紫色黑水晶核心；手持黑色長法杖，杖頂嵌尖銳紫色水晶。
中性的戰鬥預備站姿，身體微微低重心，法杖垂直握在身側，另一手自然抬起準備施法；臉和視線朝向畫面左側，雙腳穩定站立，腳底水平對齊。
```

首幀動作：

- `idle_0`：獨立的中性呼吸基準姿勢，不直接重用 seed。
- `hurt`：上半身微幅後仰，呈現清楚但不誇張的受擊反應。
- `cast`：施法手朝畫面左側伸出，法杖穩定持於身側；不含紫色符文或其他魔法特效。
- `death`：單膝跪地、頭部低垂，開始倒下但仍保留角色輪廓與法杖辨識度。
- `attack`：低重心蓄力，法杖向後斜舉，紫色魔力在杖頂聚集，作為攻擊起手幀。

- `attack`：5 幀，`foe_necro_attack_0..4.png`；涵蓋起手、蓄力、揮出、命中頂點與收招。
