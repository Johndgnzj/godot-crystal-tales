# 角色設定（Character Bible）

- 狀態: 主角＋NPC 已整理；反派內在待定
- 最後更新: 2026-07-17
- 資料來源: `resources/content/content.json` 的 `party[].story`／`dialogue.json` 的 DLG。角色數值（base/growth/裝備）以 `.tres` 為準，本檔只記敘事面。改角色台詞/背景後回頭同步本檔。

> 依 [劇本寫作心法](../劇本寫作心法.md) 雨情原則：**主角必須真實＝有缺點**；反派人性化更好。整理時特別標出每個主角的「缺點」與「核心動機」。

## 一、可操作主角

### 路德（ludo）— 主角
- **設定**：芳蕾鎮鐵匠漢克與瑪莎的獨生子，15 歲。`class=explorer`，主屬性 str。
- **性格/缺點**：從小聽旅人的冒險故事長大，嚮往外面世界；**行動總比思考快半拍**（缺點），闖禍後道歉速度全鎮最快。
- **核心動機**：偷跟亞倫進礦山 → 亞倫為掩護他失蹤 →「我要成為真正的冒險者，查出那一天的真相。」
- **關鍵設定**：身上有**蓋婭女神的記號**，會隨冒險越來越亮（主線最大伏筆，`【待拍板】`）。

### 瑪琳（marin）— 女主角/青梅竹馬
- **設定**：旅店老闆娘朵拉的女兒，與路德一起長大。`class=explorer`，主屬性 agi。
- **性格/缺點**：冷靜務實，是路德闖禍前的「最後一道煞車」——**雖然常常煞不住**（她的可愛缺點）。
- **核心動機**：礦山意外後，看著把自己關在房裡的路德，決定同行——「要去就一起去。你一個人，我不放心。」
- **關鍵物**：母親朵拉在她出發前偷偷塞進行囊的瑪瑙護符。

### 亞倫（aaron）— 恩師（序章 guest，之後失蹤）
- **設定**：借宿芳蕾鎮的 **A 級冒險者**，`class=veteran`，Lv20，`guest=True`（序章限定同行）。話少，劍快。
- **來歷**：成謎。只知他一直在調查北方礦山的異變。看出路德眼裡的光「和年輕時的某人很像」（伏筆：那個人是誰？）。
- **退場**：礦山深處遇魔影，擋在路德身前——「快跑！把這裡的事告訴鎮上的人！」此後下落不明。他留下的劍術與背影成了路德的目標。
- **`【待拍板】`**：生死未定；是否與「礦道底下的源頭」有關（mine_after 已埋此問）。
